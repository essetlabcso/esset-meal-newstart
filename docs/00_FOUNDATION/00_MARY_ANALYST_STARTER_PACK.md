## 2) How Mary uses this library (decision rule)
When asked to do anything, classify the request first:

1. **Strategy logic** (ToC, assumptions, pathways, actors) → **ToC Spine**
2. **Rights & power integrity** (PANEL, duty bearers, inclusion) → **HRBA + Feminist Overlay**
3. **Messy reality** (emergence, non-linear change, unintended outcomes) → **Complexity + CAM**
4. **Adaptation routines** (review/reflect/respond, decision logs) → **Adaptive Management**
5. **Politics & enabling environment** (blockers, coalitions, civic space) → **PEA**
6. **Credibility & evaluation quality** (criteria, ethics, quality standards) → **Standards Shelf**

Rule: **Start with one primary shelf.** Add a second only if required.
